# Keyboard Shortcuts

Master these shortcuts for a faster workflow.

## Tool Selection

| Key | Tool |
|-----|------|
| `V` | Select mode |
| `1` | Rectangle |
| `2` | Pill |
| `3` | Diamond |
| `4` | Circle |
| `5` | Container |
| `6` | Text |
| `7` | Arrow/Line |

## Actions

| Shortcut | Action |
|----------|--------|
| `A` | Auto-arrange nodes |
| `M` | Open Mermaid import |
| `G` | Toggle grid snap |

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Redo (alternative) |
| `Ctrl+C` | Copy selected |
| `Ctrl+V` | Paste |
| `Ctrl+A` | Select all |
| `Delete` | Delete selected |
| `Backspace` | Delete selected |
| `Escape` | Deselect all / Cancel operation |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Scroll wheel` | Zoom in/out |
| `Middle-click drag` | Pan canvas |
| `Click + drag` (empty space) | Pan canvas |

## Selection

| Shortcut | Action |
|----------|--------|
| `Click` | Select single node |
| `Ctrl+Click` | Add to selection |
| `Drag` (empty space) | Marquee select |
| `Shift+Click` | Select connection |

## Text Editing

| Shortcut | Action |
|----------|--------|
| `Double-click` | Edit node label |
| `Enter` | Confirm edit |
| `Escape` | Cancel edit |

## Tips

- Press number keys `1`-`7` to quickly switch tools while drawing
- `Escape` is your friend - cancels most operations
- Copy/paste works across browser tabs with the same Draph instance
