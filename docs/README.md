## Quick Start

1. Open [draph.sanath.dev](https://draph.sanath.dev)
2. Pick a shape tool (`1`-`7`) and draw on the canvas
3. Connect nodes by dragging from edge connection points
4. Style with the floating toolbar
5. Share the URL or export as PNG/SVG
